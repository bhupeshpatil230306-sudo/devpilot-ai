# 🚀 DevPilot AI

> An AI-powered DevOps Assistant that helps developers generate scripts, Dockerfiles, Kubernetes YAML, CI/CD pipelines, Linux commands, and more using Generative AI.

---

## 📌 Overview

DevPilot AI is an intelligent DevOps assistant built to simplify common DevOps tasks. Users can select a tool, enter a prompt, and instantly receive AI-generated solutions powered by Google's Gemini API.

---

## ✨ Features

- 🤖 AI-powered DevOps assistance
- 🐳 Dockerfile generation
- ☸️ Kubernetes YAML generation
- ⚙️ CI/CD Pipeline generation
- 💻 Linux command generation
- 📜 Shell script generation
- 📝 Markdown formatted responses
- 📋 Copy generated output
- 📥 Download generated output
- ⚡ Fast and responsive interface

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/bhupeshpatil230306-sudo/devpilot-ai.git
cd devpilot-ai
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file inside the `backend` folder.

```
GEMINI_API_KEY=YOUR_API_KEY
```

> **Never commit your `.env` file to GitHub.**
